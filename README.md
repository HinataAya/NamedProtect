NamedProtect

[Minecraft][Bukkit] Plugin.

用命名牌保護你的生物!

Protect your entities by NameTag!

不需使用指令，只要在名字後加上特定標籤即可。

 No command, just add a specail tag on the end of NameTag.

被動型及可馴服的生物都可以受到保護。

Passive and tamable entities can be protected.
